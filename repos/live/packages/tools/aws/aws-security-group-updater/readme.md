# aws-security-group-updater

## Install

```
npm i -g aws-security-group-updater
```

## Usage

```js
aws-sgu updated

aws-sgu update
```

## Development

```
npm i
npm run build
```

## Publish

Use [Lerna](https://github.com/lerna/lerna) in the parent directory.
